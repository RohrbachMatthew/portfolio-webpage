TECHbuy E-commerce Platform

TECHbuy Github Link

TECHbuy is an online business platform developed collaboratively to sell tech products. The website offers a seamless shopping experience for customers, featuring product browsing, detailed views, and secure checkout. The project utilizes HTML, CSS, JavaScript for the frontend, Python (Flask) for the backend, and MySQL for the database.

Techbuy Database Administration Tool

This repository includes a database administration program for viewing, filtering, entering, editing and backing up data.
